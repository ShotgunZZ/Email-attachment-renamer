Test License Key Generation

This page is for testing the license key generation without going through Stripe checkout.

Click "Generate Test License" to create a test license key.

How to use this test key:

  1. Click "Generate Test License" to create a test license key
  2. Copy the generated license key
  3. Open the Gmail Attachment Renamer options page
  4. Go to the "License" tab
  5. Paste the license key and click "Activate License"
